## Hostel management

A Project as Assignment for Mymensingh Polytechnic Institute.

### Setup

clone this repo and run following commands:

```sh
composer install
```

```sh
cp .env.example .env
```

```sh
php artisan key:generate
```

Installing node dependencies:

```sh
pnpm install
```

Start development server:

```sh
php artisan serve
```

AND

```sh
pnpm dev
```
